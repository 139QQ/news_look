# 数据库工具

此目录包含与数据库操作相关的工具脚本：

- `check_db.py` - 检查数据库状态和内容
- `database.py` - 数据库操作的核心功能
- `fix_db_names.py` - 修复数据库文件名编码问题
- `fix_keywords.py` - 修复数据库中的关键词

## 使用方法

### 修复数据库文件名
```bash
python fix_db_names.py
```

### 检查数据库
```bash
python check_db.py
```

### 修复关键词
```bash
python fix_keywords.py
```
